# Инструкции за автоматизирани UI тестове с Playwright

1. Инсталирай Playwright (ако не е инсталиран):
   ```
   npm install --save-dev @playwright/test
   npx playwright install
   ```

2. Увери се, че локалният сървър работи на http://localhost:3000
   ```
   npm run dev
   ```

3. Стартирай тестовете:
   ```
   npx playwright test tests/app.spec.js
   ```

Тестът ще:
- Зареди приложението
- Кликне на "Генерирай меню"
- Изчака менюто да се появи
- Навигира между табовете
- Провери, че няма видима грешка

Ако има грешки или тестът не минава, ще видиш подробна информация в конзолата.
